Motivation

I started this project when I was preparing for academic job applications in US & Canada.
This was a way to find a little more information about cities in bulk, so I could choose which places I wanted to focus on.
The main goal of this project was to provide a single API that could easily import all this data into Google Sheets.

Data Sources

  • weather data: meteostat-python
    • Md. = median;
    • Avg. = average;
  • livability: areavibes.com
  • population: areavibes.com
  • large cities (def: cities with a population of 100k+ (US) or 50k+ (Canada)):
    Sources accessed on 2022-01-27:
