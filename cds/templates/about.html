Motivation

I started this project when I was looking into universities in US & Canada.
This was a way to find a little more information about cities in bulk, so I could choose which places I wanted to focus on.
The main goal of this project was to provide a single API that could easily import all this data into Google Sheets.

Data Sources

  • Basic city info: http://geonames.org/
  • City photo: Google Places API
  • Weather: meteostat-python
    • Md. = median;
    • Avg. = average;
  • City livability: https://areavibes.com/
  • Major cities: https://worldpopulationreview.com/world-cities (accessed on 2022-01-28)
  • Urban area data: https://teleport.org/
